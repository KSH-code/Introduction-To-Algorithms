# 지수 함수
모든 실수 a > 0,m,n에 대해 다음 등식이 성립한다.

a^0 = 1

a^1 = a

a^-1 = 1/a

(a^m)^n = a^(mn)

(a^m)^n = (a^n)^m

(a^m)(a^n) = a^(m+n)
